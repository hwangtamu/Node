[Tutorial 3] Launch your first Rails app on Cloud9

This tutorial aims to help you start your Homework#2.

 

Step 1: Create a new Cloud9 environment and install Ruby 2.4.0 as in Tutorial 1.

 

Step 2: Download the homework repo from Github and cd into the repo.

 

$ git clone https://github.com/saasbook/rottenpotatoes-rails-intro.git
$ cd rottenpotatoes-rails-intro

Step 3: Install Rails.

 

$ sudo su
$ rvm use 2.4.0
$ gem install rails

Step 4:  Run "bundle install".

 

$ bundle install --without production
$ exit

Step 5: Create database and start the app.

 

$ bundle exec rake db:setup
$ rails server -p $PORT -b $IP

Step 6: Click on the link in the pop-up dialogue (near top right corner of the IDE)  to view the app.

 

 

Now you have started your rottenpotatoes application.
